BugOzon01 - SCRUM-16

Summary: Поиск с пустым запросом в выбранной категории не отображает товары категории

Priority: Medium

Description:



Steps:

1\. Открыть ozon.ru;

2\. В поисковой строке выбрать категорию (например, «18+» или «Канцелярские товары»);

3\. Нажать поиск с пустым полем.



Actual Result: Поиск не выполняется, страница остаётся без изменений.



Expected Result: Должны отобразиться товары выбранной категории.





BugOzon02 - SCRUM-17

Summary: При поиске в категории 18+ предложения нерелевантны, без выбора варианта поиск не выполняется

Priority: Medium

Description:



Steps:

1\. Открыть ozon.ru;

2\. Выбрать категорию «18+»;

3\. Ввести букву «А»; 4. Нажать поиск.



Actual Result: Поиск не выполняется, появляется выпадающий список с нерелевантными предложениями (например, «Автошампунь»).



Expected Result: Должны отобразиться релевантные предложения, относящиеся к выбранной категории, либо поиск должен выполняться без выбора.





BugOzon03 - SCRUM-18

Summary: После выбора предложенного варианта в категории 18+ категория поиска сбрасывается на «Везде»

Priority: High

Description:



Steps:

1\. Открыть ozon.ru;

2\. Выбрать категорию «18+»;

3\. Ввести «А» и выбрать «Автошампунь»;

4\. Подтвердить возраст;

5\. Посмотреть на строку поиска.



Actual Result: Категория сбрасывается на «Везде», поиск выполняется по всему сайту.



Expected Result: Категория должна остаться «18+» (или измениться на релевантную, например «Автотовары»), поиск должен выполняться в контексте категории.
