Поиск:
Поиск по существующему товару — результаты отображаются.
Поиск по несуществующему товару — сообщение «Ничего не найдено», предложены аналоги.
Поиск с пустым запросом в выбранной категории — показываются товары категории / (баг) = не дает сделать поиск с пустым значением, а должен перекинуть в выбранную категорию.
Поиск с первой буквой в категории 18+ — предлагаются релевантные варианты / (баг) = не выдает релевантных запросов, а без них не делает поиск.
Поиск с выбором предложенного варианта — категория не сбрасывается / (баг) = Сбрасывается выбранная категория.
Поиск с длинным запросом (>50 символов) — корректная обработка.
Поиск со спецсимволами (!@#$) — корректная обработка или экранирование.
Переключение категории в поиске - показываются товары новой категории. / (баг) = не показывает товары новой категории.


Фильтры и сортировка:
Фильтр по цене — обновление списка.
Фильтр по сроку доставки («сегодня», «завтра») — корректная фильтрация.
Фильтр по бренду — обновление списка.
Сортировка «по популярности» (по умолчанию) — работает.
Сортировка «по цене» — работает.
Скидки недели - отображаются товары по "скидке недели" / (баг) = отображаются товары НЕ только со "скидкой недели".

Карточка товара:
Отображение фото, цены, названия, рейтинга.
Кнопка «В корзину» активна, если товар в наличии.
Для товара не в наличии — кнопка «Похожие» или уведомление.
Изменение количества товара в карточке перед добавлением.
При переходе в карточку товара, отображает ближайший склад с наличием

Корзина
Добавление товара — появляется уведомление, счётчик обновляется.
Изменение количества товара в корзине — пересчёт суммы.
Удаление товара из корзины — товар исчезает, сумма обновляется.
Переход в корзину из любой точки сайта (кнопка вверху).
Пустая корзина — отображается сообщение "Корзина пуста"
Наличие скидок — корректное отображение итоговой суммы.
